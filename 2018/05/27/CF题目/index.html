#A Twisty Movement

C. A Twisty Movement

time limit per test：1 second

memory limit per test：256MB

A dragon symbolizes wisdom, power and wealth. On Lunar New Year’s Day, people model a dragon with bamboo strips and clothes, raise them with rods, and hold the rods high and low to resemble a flying dragon.

A performer holding the rod low is represented by a 1, while one holding it high is represented by a 2. Thus, the line of performers can be represented by a sequence a1, a2, …, a**n.

Little Tommy is among them. He would like to choose an interval [l, r] (1 ≤ l ≤ r ≤ n), then reverse a**l, a**l + 1, …, a**r so that the length of the longest non-decreasing subsequence of the new sequence is maximum.

A non-decreasing subsequence is a sequence of indices p1, p2, …, p**k, such that p1 < p2 < … < p**k and a**p1 ≤ a**p2 ≤ … ≤ a**p**k. The length of the subsequence is k.

Input

The first line contains an integer n (1 ≤ n ≤ 2000), denoting the length of the original sequence.

The second line contains n space-separated integers, describing the original sequence a1, a2, …, a**n (1 ≤ a**i ≤ 2, i = 1, 2, …, n).

Output

Print a single integer, which means the maximum possible length of the longest non-decreasing subsequence of the new sequence.

#题解#

这道题是求一个数组A中只包含1、2两种数字，求出一个区间，将这个区间翻转后，数组A为最长不下降子序列。

这道题我换了很多思路才做出来。最后的算法时间复杂度为O(N2)O(N2).

首先肯定要找出数组最大旋转价值的区间旋转即可。

定义左右区间的旋转价值：例如果左侧区间每出现一个2则值得旋转，旋转价值+1，如果出现一个1则不值得旋转，旋转价值就-1，右侧则与左侧对称。

首先符合题意的区间xy（0<= x < y <=len）有这个特点，把区间以i划分为左右两个部分(0<x<i<y<len)，假设答案的i值a，那么x到a的旋转价值是所有x中最高的，相同，y的旋转价值也是从a到len最高的。

那么如何找到答案的i？在所有可能的区间xy中，只要保证区间xy左右旋转价值尽量的大和保证x到a和a到y的数字2、1旋转价值的差距尽量的小就可以。我们只要从每个区间xy中，找出一个区间中的其保证了左右两个区间最大的旋转价值后取其左右两个其中较小旋转价值是所有可能的xy中与上述等价的条件中最大的就可以保证该区间是所有左右旋转价值差最小的 [1]。
